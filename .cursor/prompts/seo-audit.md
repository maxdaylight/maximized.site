# SEO Audit

Audit the SEO of: [filename.html or "all pages"]

Check for:
1. Unique <title> in "Page | Maximized.site" format
2. Meta description exists, unique, 150-160 chars
3. All Open Graph tags present and populated
4. canonical <link> present and correct URL
5. One <h1>, correct heading hierarchy
6. All <img> have descriptive alt text
7. JSON-LD structured data present (Person on index, BreadcrumbList on interior)
8. Internal links use relative paths
9. Page listed in sitemap.xml
10. robots.txt allows this page

Report findings and fix all issues found.
